## GDMap

### 属性

| 属性       | 说明                | 类型             | 默认值                |
| ---------- | ------------------- | ---------------- | --------------------- |
| address    | 地址                | `string`         | 浦东新区纳贤路 800 号 |
| title      | 标题                | `string`         | 地图                  |
| mapStyle   | 列表 class          | `css`            | -                     |
| titleStyle | 标题 class          | `css`            | -                     |
| custom     | 是否自定义地图      | `boolean`        | -                     |
| option     | 自定义地图的 option | `见高德地图配置` | -                     |
| init       | 地图初始化方法      | `(map) => {}`    | -                     |
