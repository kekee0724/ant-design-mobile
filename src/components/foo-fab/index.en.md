### Fab

| 属性         | 说明                                              | 类型     | 默认值         |
| ------------ | ------------------------------------------------- | -------- | -------------- | --- |
| position     | 位置： 'right-bottom'、'right-top'、'left-top'... | `string` | 'right-bottom' | -   |
| text         | 内容                                              | `any`    | -              |
| --fab-margin | style 属性                                        | `style`  | 16px           |

### FabButtons

| 属性     | 说明             | 类型     | 默认值 |
| -------- | ---------------- | -------- | ------ |
| position | 位置： 'left'... | `string` | 'top'  |

### FabButtons

| 属性  | 说明 | 类型     | 默认值 |
| ----- | ---- | -------- | ------ |
| label | 提示 | `string` | -      |
