### 属性

| 属性        | 说明         | 类型                  | 默认值 |
| ----------- | ------------ | --------------------- | ------ | --- |
| title       | 标题         | `string \| reactNode` | -      |
| showText    | 展开的描述   | `string \| reactNode` | 展开   |
| hideText    | 隐藏的描述   | `string \| reactNode` | 收起   |
| children    | 标题         | `string \| reactNode` | -      |
| defaultShow | 默认是否显示 | `string \| reactNode` |        | -   |
